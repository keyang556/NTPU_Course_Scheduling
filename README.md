**作者：黃振彰**

## 專案簡介

這是一個純前端的通識課程試排工具，專為國立臺北大學學生設計。  
幫助你在正式選課前模擬志願順序、檢查時間衝突、避開不想上的時段，並即時確認課程組合是否可行。

支援手機與電腦，內建深色模式與一鍵分享功能。

## 主要功能

- 關鍵字搜尋課程名稱或老師
- 依向度篩選、依向度或時間排序
- 加入志願後可拖曳調整優先順序
- 時間網格顯示已選課程，並可點擊格子屏蔽時段
- 方格／列表兩種顯示模式
- 顯示剩餘名額、可加簽、授課語言、備註等完整資訊
- 一鍵 Google 搜尋課程評價
- 志願順序自動寫入網址，「分享」按鈕可複製連結給同學，直接還原相同排序

## 如何使用

1. 若有不能上的時段，點擊上方時間網格的格子進行屏蔽
2. 在搜尋欄輸入課程名稱或老師，或利用**向度篩選**功能找課
3. 找到後可先點擊「評價」按鈕查詢評價
4. 點擊「＋加入」
5. 「志願排序」區域用拖曳調整優先順序
6. 確認沒問題後，點右上角「分享」複製連結備用

課程資料來自國立臺北大學課程查詢系統（通識課程頁面）：

## 注意事項

- 本專案大部分程式碼與開發內容由 AI 協作生成。
- 試排結果不代表實際選課結果，請務必以學校教務系統公告為準。
- 目前手機網頁版之適配體驗尚未完善，建議使用電腦版以獲得最佳操作體驗。
